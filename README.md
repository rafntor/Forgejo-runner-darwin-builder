# Forgejo-runner-windows-built

This repository contains build scripts for creating Forgejo-runner windows binaries.
The binaries are available on the [release page](https://github.com/Crown0815/forgejo-runner-windows-built/releases).

The original deprecated repository can be found [here](https://github.com/Crown0815/forgejo-runner-windows).
The original code for the Forgejo-runner is available [here](https://code.forgejo.org/forgejo/runner).
